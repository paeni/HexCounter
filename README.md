# 16進数カウンター

JavaScriptだけで動作する静的な16進数カウンターです。

## 機能

- 1ずつ増減
- `0x10`（16）ずつ増減
- 10進数と16進数の同時表示
- 任意の値を入力
- リセット
- キーボードの上下キーで増減

## GitHub Pagesで公開する方法

1. このフォルダ内のファイルをGitHubリポジトリへ追加します。
2. GitHubのリポジトリ画面から `Settings` を開きます。
3. `Pages` を開きます。
4. `Deploy from a branch` を選びます。
5. `main` ブランチと `/ (root)` を選択して保存します。
